Dasar Penggunaan Flexbox

Part 1 - Halaman 1 dari 2

Flexbox merupakan salah satu jenis nilai dari property display pada CSS (Penggunaannya "display: flex")

Flexbox berguna untuk mempermudah kita memanipulasi tata letak (layout) suatu element web

Menggunakan flexbox sangatlah mudah:

  1. Siapkan minimal dua element. Dua/lebih element tersebut harus memiliki hubungan parent-child (orang tua-anak)
  2. Terapkan aturan css display flexbox ke element parent

Contoh:

<div style="display: flex;"> <- parent element
  <div style="background: red">Elemen Anak</div> <- child element
  <span style="background: blue">Elemen Anak ke-2</span> <- child element juga
</div>

Hasil:

Elemen Anak
Elemen Anak ke-2

Dengan demikian, sekarang tata letak child element sudah dikendalikan oleh parent.

Tata letak yang diberikan secara default oleh flexbox adalah berupa elemen yang sebaris/sejajar.

Buktinya, di bawah ini adalah hasilnya jika kita tidak menerapkan display: flex ke parent-nya:

Elemen Anak
Elemen Anak ke-2
